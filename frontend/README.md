# J-ELECTRE Frontend

This is a Vite + React frontend scaffold for the J-ELECTRE API. It uses Material UI for controls, Axios for HTTP requests, and Recharts for plotting (charts can be added in GraphView components).

Quick start:

1. cd frontend
2. npm install
3. npm run dev

By default the app expects the API to be available on the same origin. When running the Python FastAPI server on another port, configure a proxy or set axios baseURL in the code.
